## Credit Card Approval Prediction System

The Credit Card Approval Prediction System uses machine learning to automate the credit card approval process in the banking sector. It analyzes applicant details such as income type, employment status, family status, housing type, and credit history to predict approval or rejection.

## Machine Learning Pipeline

The project follows a complete machine learning workflow, including:
- Data collection  
- Data preprocessing  
- Data visualization  
- Feature engineering  
- Categorical encoding  
- Model training and testing  
- Performance evaluation  

## Models Used

Various classification algorithms are implemented and compared:
- Logistic Regression  
- Decision Tree  
- Random Forest  
- XGBoost  

The best-performing model is selected based on accuracy and efficiency.

## Deployment

The final model is integrated into a **Flask-based web application**, allowing users to:
- Enter applicant details  
- Get instant credit card approval prediction  

The system is also deployed using **IBM Watson Machine Learning**, enabling cloud-based and scalable real-time predictions.

## Conclusion

This project demonstrates practical experience in:
- Machine Learning  
- Data Analysis  
- Flask Web Development  
- Cloud Deployment  

It effectively solves a real-world banking problem by improving the speed and accuracy of credit card approval decisions.
